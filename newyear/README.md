# 春节贺岁页面 - 龙年大吉

一个酷炫的前端页面，用于庆祝2025年春节（龙年）。页面包含丰富的动画效果、交互功能和春节元素。

## 功能特性

### 🎨 视觉设计
- 传统春节红色和金色主题
- 动态背景装饰（灯笼、福字、龙元素）
- 响应式设计，适配各种设备

### ⏰ 核心功能
- **春节倒计时**：实时显示距离2025年春节的剩余时间
- **祝福语轮播**：自动切换春节祝福语，支持手动切换
- **烟花系统**：Canvas实现的粒子烟花效果
- **交互元素**：
  - 点击灯笼播放音效和粒子效果
  - 燃放烟花按钮
  - 新春音乐播放控制

### 🎮 交互体验
- 鼠标移动触发随机烟花
- 键盘快捷键支持
- 触摸设备优化

## 技术栈

- **HTML5**：语义化标签，Canvas API
- **CSS3**：Flexbox/Grid布局，CSS动画，渐变，媒体查询
- **JavaScript (ES6+)**：面向对象编程，事件处理，Canvas绘图
- **外部资源**：
  - Font Awesome图标
  - Google Fonts中文字体
  - Mixkit音效和音乐

## 使用方法

1. 直接使用浏览器打开 `index.html` 文件
2. 或者使用本地服务器运行：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

3. 访问 `http://localhost:8000` 查看页面

## 键盘快捷键

- **空格键**：播放/暂停新春音乐
- **方向键右键**：切换到下一条祝福语
- **F键**：燃放烟花
- **L键**：点亮灯笼（播放音效和粒子效果）

## 开发说明

### 主要动画效果
1. **灯笼摆动**：CSS关键帧动画
2. **福字飘落**：CSS动画延迟和变换
3. **烟花粒子**：Canvas粒子系统
4. **倒计时跳动**：CSS缩放动画

### 交互逻辑
1. **倒计时更新**：每秒计算剩余时间
2. **祝福语轮播**：5秒自动切换，支持点击切换
3. **烟花系统**：粒子物理模拟（重力、衰减）
4. **音频控制**：处理浏览器自动播放策略

## 自定义修改

### 修改春节日期
在 `script.js` 文件中修改 `springFestivalDate` 变量：
```javascript
const springFestivalDate = new Date('2025-01-29T00:00:00');
```

### 修改祝福语
在 `index.html` 中修改祝福语卡片内容。

### 修改颜色主题
在 `style.css` 中修改CSS变量：
```css
:root {
    --primary-red: #d32f2f;
    --primary-gold: #ffd700;
}
```
